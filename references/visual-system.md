# Visual System

## Direção visual

A apresentação segue uma estética executiva cinematográfica.

O objetivo não é parecer um dashboard comum ou um PowerPoint corporativo tradicional.

A sensação visual precisa transmitir:

- sofisticação;
- controle operacional;
- leitura premium;
- clareza executiva;
- ritmo visual;
- foco narrativo.

O sistema visual mistura:

- keynote cinematográfico;
- dashboards minimalistas;
- interfaces premium;
- storytelling de growth/revenue.

## Layout

Padrão fixo:

- 16:9 widescreen;
- grandes áreas de respiro;
- grid consistente;
- títulos fortes;
- poucos elementos simultâneos;
- leitura hierárquica clara.

Evitar:

- excesso de cards;
- excesso de texto;
- bullets gigantes;
- visual varejista;
- excesso de bordas.

## Tipografia

Estrutura recomendada:

- títulos: fonte sans-serif premium bold;
- subtítulos: medium/semi-bold;
- conteúdo: regular com alto espaçamento;
- números: monoespaçados opcionais para métricas.

Fontes compatíveis:

- Inter
- Geist
- Satoshi
- SF Pro
- General Sans

## Paleta visual

Base principal:

- fundos escuros ou neutros;
- branco/off-white para contraste;
- tons suaves de cinza;
- cor de destaque apenas para narrativa.

Evitar:

- múltiplas cores competindo;
- gradientes exagerados;
- saturação alta.

## Motion

As animações devem parecer fluidas e invisíveis.

Preferências:

- fade;
- translate suave;
- blur leve;
- stagger discreto;
- micro-delay.

Evitar:

- bounce;
- zoom agressivo;
- animações caricatas;
- excesso de rotação.

## Narrativa visual

Cada slide precisa responder:

1. O que aconteceu?
2. Por que isso importa?
3. Qual decisão isso influencia?

A apresentação deve funcionar como leitura executiva, não como decoração.

## Estrutura cinematográfica

A progressão visual ideal:

1. abertura forte;
2. contexto;
3. problema;
4. dados;
5. interpretação;
6. impacto;
7. próximos passos.

## Padrão executivo

O slide deve parecer:

- produto premium;
- sala de conselho;
- relatório de growth de empresa de tecnologia;
- leitura financeira moderna.

Nunca parecer:

- apresentação escolar;
- template genérico do Canva;
- dashboard poluído.
